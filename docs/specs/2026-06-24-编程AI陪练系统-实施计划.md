## Phase 1：项目骨架 + 对话陪练 MVP

**目标**：Electron 桌面壳可运行，基础对话面板可用，陪练Agent 能答疑

### 1.1 项目初始化

| 任务 | 产出 |
|------|------|
| Electron + React + TypeScript 脚手架 | `package.json`, `tsconfig.json`, `electron/main.ts` |
| Vite 构建配置（主进程 + 渲染进程） | `vite.config.ts` |
| Python FastAPI 后端项目初始化 | `backend/pyproject.toml`, `backend/main.py` |
| Electron 启动时自动拉起 Python 子进程 | `electron/services/pythonBridge.ts` |
| 前后端 WebSocket 通信打通 | `frontend/hooks/useWebSocket.ts` |

### 1.2 对话面板

| 任务 | 产出 |
|------|------|
| 聊天 UI 组件（消息列表 + 输入框 + Markdown 渲染） | `frontend/components/ChatPanel/` |
| 代码块高亮渲染 | 集成 `react-syntax-highlighter` |
| 流式输出支持 | SSE/WebSocket streaming |

### 1.3 陪练Agent

| 任务 | 产出 |
|------|------|
| LangChain ChatOpenAI 接入（支持本地 Ollama + 云端 API 切换） | `backend/agents/base.py` |
| 陪练Agent 核心逻辑（答疑/代码审查/Debug） | `backend/agents/coach_agent.py` |
| Agent 与前端对话联通 | `backend/api/chat.py` |

### Phase 1 完成标准
- 打开桌面应用 → 看到对话界面 → 输入编程问题 → AI 流式回答 → 代码块正确高亮

## Phase 2：评估 + 教学 + RAG 知识库

**目标**：用户能完成摸底评估，生成学习计划，AI 教学有知识库支撑

### 2.1 RAG 基础设施

| 任务 | 产出 |
|------|------|
| ChromaDB 初始化 + 连接管理 | `backend/rag/vector_store.py` |
| bge-large-zh 本地 Embedding 服务 | `backend/rag/embedding.py` |
| 文档分块 + 索引流水线 | `backend/rag/indexer.py` |
| 混合检索（向量 + BM25）+ 重排序 | `backend/rag/retriever.py` |
| 初始知识库：Python 官方文档 + JS MDN 精选 | `data/knowledge/` |

### 2.2 评估Agent

| 任务 | 产出 |
|------|------|
| 自适应出题引擎（难度自适应算法） | `backend/agents/eval_agent.py` |
| 评估 UI（选择题 + 编程题 + 结果雷达图） | `frontend/components/Evaluation/` |
| 能力画像生成 + 存储 | `backend/models/user_profile.py` |

### 2.3 教学Agent + 学习计划

| 任务 | 产出 |
|------|------|
| 计划生成引擎（基于画像 + 目标） | `backend/agents/teach_agent.py` |
| 学习计划 UI（阶段视图 + 每日任务） | `frontend/components/LearningPlan/` |
| 对话式教学（RAG 增强讲解） | 陪练Agent 升级 |

### 2.4 编排器基础版

| 任务 | 产出 |
|------|------|
| LangGraph StateGraph 定义 | `backend/orchestrator/graph.py` |
| 意图路由（评估/教学/陪练分流） | `backend/orchestrator/router.py` |

### Phase 2 完成标准
- 新用户 → 评估 → 雷达图 → 自动生成学习计划 → 开始按计划学习 → AI 教学引用知识库内容
